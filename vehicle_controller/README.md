# vehicle_controller
Vehicle controller package

## Overview
本パッケージは対向２輪型の車体動作を制御する「vehicle_controller」ノードが含まれています。
対向２輪型とは２個の車輪を左右対称にロボットに取り付け、それぞれを独立に駆動するようにしたものです。
